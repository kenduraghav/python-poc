=== Variables
- Dynamically typed: no need to declare the type.

=== Common Data Types
- `int`, `float`, `str`, `bool`

=== Input
[source,python]
----

age = int(input("Enter your age: "))
----

=== Output
[source,python]
----

print(f"You are {age} years old")
----

== âš–ï¸ Control Statements: `if`, `for`, `while`, `break`, `continue`

=== if-else
[source,python]
----
if age >= 18:
    print("Adult")
elif age > 12:
    print("Teenager")
else:
    print("Child")
----

=== Loops

==== for loop
[source,python]
----
for i in range(5):
    print(i)
----

==== while loop
[source,python]
----
count = 0
while count < 5:
    print(count)
    count += 1
----

=== break and continue
[source,python]
----
for i in range(10):
    if i == 5:
        break
    if i % 2 == 0:
        continue
    print(i)
----

== ðŸ”¢ Functions

=== Defining and Using Functions
[source,python]
----

----

=== Ternary Expression
[source,python]
----
def check_even_odd(num):
    return "Even" if num % 2 == 0 else "Odd"
----

== ðŸ”Ž Dictionaries

=== Definition and Access
[source,python]

=== Nested Dictionary Example
[source,python]
----
def create_student(id, name, age, subjects):
    return {id: {"id": id, "name": name, "age": age, "subjects": subjects}}

students = {}
students[1] = create_student(1, "Alice", 22, ["Math", "Science"])[1]
----

== âœï¸ Coding Standards & Best Practices

- Use `snake_case` for function names (e.g., `calculate_total`)
- Use meaningful and descriptive variable names
- Comment for clarity
- Define functions before using
- Indent with 4 spaces
- Avoid magic numbers; use named constants
